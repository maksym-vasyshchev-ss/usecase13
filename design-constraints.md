# Design Constraints
The section lists the constraints accounted for in the designed solution. These can be of business, technical, resource, and other types.

N. | Description
--- | ---
CON-1 | The solution should be accessible to users as a web application
CON-2 | The web application should be accessible on mobile devices
CON-3 | Solution should be able to operate without downtime for maintenance
CON-4 | Solution should not allow data leaks
CON-5 | Calculation engine should be run as a separate microservice
CON-6 | Tax calculations should be fast under load
